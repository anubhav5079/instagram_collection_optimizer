================================================================================
KEY FILES & WHAT THEY DO
================================================================================

.env
   Contains your Instagram credentials for the Python scraper

web/.env
   Contains Instagram credentials for the Next.js app

web/vercel.json
   Tells Vercel:
   - Build command: npm run setup-db && npm run build
   - Output directory: out (static files)
   - Environment variables needed: INSTAGRAM_* variables

web/package.json
   Specifies:
   - All npm dependencies (next, react, better-sqlite3, etc.)
   - Build scripts:
     * setup-db: Initializes database
     * dev: Runs development server
     * build: Builds Next.js app
     * start: Starts production server

web/scripts/setup-db.js
   Runs during build to:
   - Create data directory
   - Initialize SQLite database
   - Create tables from schema
   - Verify setup is complete

web/src/lib/db.ts
   Core database module:
   - Exports functions to get collections and posts
   - Has error handling with try-catch
   - Falls back to demo data if database unavailable

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: Build fails during deployment
FIX: Check Vercel logs. Most common: wrong environment variables or missing files

ISSUE: "Could not locate bindings file" error
FIX: This is normal! The app falls back to demo data. No action needed.

ISSUE: Site shows blank page
FIX: Check browser console (F12) for errors. Check Vercel build logs.

ISSUE: Environment variables not working
FIX: Verify they're set in Vercel project settings (Settings → Environment Variables)

ISSUE: Database not initializing
FIX: Check that setup-db.js exists in web/scripts/ and vercel.json includes setup-db in buildCommand

ISSUE: Can't connect to GitHub
FIX: Ensure you've granted Vercel permission to access your GitHub account

ISSUE: Deployment is very slow
FIX: This is normal first time. Subsequent builds are faster.
